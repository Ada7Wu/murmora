# v4 — 晨露胶片 morning dew film（视觉系统）

> 2026-05-23 ｜ 取代 v3 暮色森林绿。**仅换皮 + 字体 + 母题冷调化**：八屏流程、AI 管线、语音/TTS、logo 资产全不动。
> 落地依据两个 skill：**ui-ux-pro-max**（设计系统/UX 规则）+ **frontend-design**（避免通用 AI 风、要有明确美学方向）。

## 来源

## 设计系统（ui-ux-pro-max `--design-system` 产出 + 取舍）

- **Pattern：Minimal Single Column** —— 单列、单一主 CTA、大字、留白。与八屏「一步一屏、底部一个主按钮」天然契合，保留。
- **Style：Soft wellness（柔和助眠）** —— skill 推荐 Neumorphism，但其暗色对比低、被 skill 自身标注 ⚠；故只取「柔和圆角 + 轻投影 + 低刺激」精神，不做拟物浮雕，改走**氛围极简 + 漂浮晨雾**。
- **配色：丢弃 skill 默认薰衣草紫**（frontend-design 明确反对「紫渐变」），改用用户色卡。映射成 `:root` 语义 token：
  `--pine/--horizon/--moss/--sage/--clay`，再派生 `--text(clay) / --text-soft(sage) / --text-mute(#a7bbac) / --line / --surface / --fog`。
- **字体：Lora（衬线）+ Raleway（正文）** —— skill 为「calm/wellness/organic」推荐的组合；Lora 的 editorial 衬线气质正好呼应「胶片」感。
  字标 MURMORA、各屏标题、副标题（斜体）、降落留言用 Lora；正文/控件用 Raleway（CJK 自动回落 PingFang/雅黑）。取代原 Quicksand。

## 母题改动（v3 → v4）

- **背景**：暖森林绿渐变 → **冷松墨渐变** `#2a474d→#182b2e` + 顶部天光雾（晨陶米/ sage 低透明径向）。
- **氛围层**：原「天光雾呼吸 + 上浮孢子」→ **谷中三条横向漂浮晨雾带**（`fogdrift`，blur 大、缓慢左右游移）+ 几点冷色露珠。这是 v4 的标志性细节（frontend-design：用氛围/纹理而非纯色块）。
- **池塘母题（POND，复用 ③⑦⑧）**：几何松林 `--g` 全换冷调（松墨/远山青/苔绿）；**暖萤火（唯一暖色 #e9c46a）→ 晨露微光**（sage→苔绿的冷色露珠辉光）；水线/涟漪/苔叶转晨陶米·sage。
- **呼吸圈/降落留言**：转 sage→松墨；留言改 Lora。

## 无障碍（ui-ux-pro-max §1/§6/§7 必做项）

- **对比**：主文字 晨陶米 on 松墨 ≈ **7.5:1**（AAA）；次文字 sage ≈ 6:1；弱文字用提亮苔绿 `#a7bbac` 保 ≥4.5:1。
- **focus**：按钮加 `:focus-visible` sage 描边环（键盘可见）。
- **reduced-motion**：`@media (prefers-reduced-motion: reduce)` 全局关装饰动画/过渡。
- **font-display: swap**（Google Fonts URL 带 `&display=swap`，避免 FOIT）。

## 通道图标（去 emoji）

② 降落岛「感性/理性」原为 `◐ / 🪨` emoji，已换**几何 SVG**（`ICON_SENSORY` / `ICON_RATIONAL`）：

- **感性 = 石落涟漪**（晨陶米露珠 + sage 同心涟漪荡开）——「想说点什么」，情绪泛开。
- **理性 = 立石**（sage 棱面石 + 晨陶米内部facet线）——「有事要理清」，把一团乱理成有棱面的一块。
- 二者呼应核心隐喻「**池中之石**」，合起来即 logo（石 + 涟漪）。
- 描边走色卡 token（`.chan-ic .s`=sage 主 / `.c`=晨陶米细节 / `.sf` 淡填充）、统一 2px、`aria-hidden`（旁有文字标签，避免重复朗读）。满足 ui-ux-pro-max + frontend-design「禁 emoji 当结构图标」。
