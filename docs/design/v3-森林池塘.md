# v3 — 森林池塘视觉系统（Forest Pond）

> 当前视觉基线。**取代 v2.3「月夜深湖」**（深蓝 + 月亮 + 极光 + 山影）。
> 交互流程、八屏路由、AI 管线、语音/TTS **均不变**——本版只换"皮肤"（配色 + 母题 + 文案）。
> 配套静态草图：[`森林池塘-草图.html`](./森林池塘-草图.html)（可直接浏览器打开核对效果）。

## 1. 为什么改

用户对"月亮 + 星星"这套常见助眠意象不买账，希望要：
- **纪念碑谷（Monument Valley）那样的几何感**——平面色块拼出立体、克制、梦幻；
- **绿色**为主调；
- **"倒映着森林的池塘"** 的氛围。

落到产品灵魂仍守红线：暗色、低刺激、温柔、助眠（见 `CLAUDE.md` 设计红线）。
本版把"坠落前接住思绪"的隐喻从"月夜湖面"换成"暮色森林倒映入静池"。

## 2. 美学锚点（用 ui-ux-pro-max 风格库定位）

纪念碑谷在风格库无独立条目，本质是三者叠加：

| 风格库条目 | 取它什么 |
|---|---|
| **#12 Flat Design** | 2D 纯色块、无阴影、几何形——"平面拼立体" |
| **#42 Organic Biophilic**（森林绿系） | 绿色、自然、圆润流动、wellness 气质 |
| **#1 Minimalism / #69 Bauhaus** | 几何、克制、构成感 |
| **#65 Gradient Mesh** | 顶部柔和天光（替代极光/月晕） |

一句话：**Flat 几何松林 + 左亮右暗假立体 + 森林绿 + 镜面倒影 + 一盏暖萤火**。

## 3. 配色 token（暮色 · 提亮版）

> 经一轮"太暗"反馈后整体抬高明度到"暮色"而非"午夜"。语义 token 与 `.streamlit/config.toml` 同步。

| Token | Hex | 用途 |
|---|---|---|
| `--bg-top` 暮林天空 | `#163a2b` | 背景渐变顶（= config `backgroundColor`） |
| `--bg-bottom` | `#0e2a1f` | 背景渐变底 |
| `--water` 池水 | `#16402f` | 下半渐入水色（= config `secondaryBackgroundColor` `#1a4a37`） |
| `--f-light` 近林(亮面) | `#74c69d` | 几何树亮侧（= config `primaryColor`） |
| `--f-mid` 中林 | `#40916c` | 树身 |
| `--f-dark` 远林(暗面) | `#2d6a4a` | 暗侧 |
| `--f-deep` 最远 | `#1b4332` | 睡莲/缝隙 |
| `--mist` 天光雾 | `#b7e4c7` | 顶部辉光、水线高光、孢子 |
| `--accent` 暖萤火 | `#e9c46a` | **唯一暖色**——萤火 + 「明日第一步🌱」 |
| `--text` 主文 | `#eaf6ef` | 正文（在 `--bg-top` 上对比 ≈12:1 ✓） |
| `--muted` 辅助 | `#a9d3bd` | hint/sub（替换 v2 的 `#7e9bbd`，绿调且对比达标） |

## 4. 母题与实现（`src/app.py` 样式块）

全部纯 CSS + 既有 DOM，不加依赖、不改流程。

- **固定夜景层 `.scene`**（八屏常驻，内容之下）：顶部 `.mist` 天光雾（缓慢呼吸）+ 4 粒 `.spore` 上浮孢子。**无月、无星、无山影。**
- **`.stApp` 背景**：天光雾径向 + 林绿渐变 + 渐入池水。
- **`POND` 母题**（启动①/倾倒③/导图⑤/封存⑦ 复用）：
  - `.grove` 几何松林：5 棵 `.tree`（`clip-path` 分层三角 + `color-mix` 左亮右暗假立体，深度三档绿）。
  - `.grove.ref` **整片镜像倒影**：`scaleY(-1)` + 模糊 + 压暗(brightness .78) + 向下渐隐遮罩。
  - `.pondline` 水线高光；`.surface` 内 3 圈 `.ripple` **同心涟漪**（扁椭圆透视、缓缓外扩）+ 2 片 `.lotus` **睡莲叶**。
  - `.firefly` **暖萤火**：低悬林梢、上浮 + 呼吸辉光，倒影中可见。
- **呼吸圈 `.breath`**（归位④/降落⑧）：渐变改 `#b7e4c7→#1b4332`，节奏不变（4-1-6）。
- **字体**：标题 `MURMORA` 用 Google Font **Quicksand**（圆润几何）；中文回退系统字。
- **可达性**：保留 `@media (prefers-reduced-motion)`（草图已带；睡前 + ADHD 低刺激尤其重要——后续在 app 内补全见 TODO）。
